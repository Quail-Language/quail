object _get(string key)

Non-static May be null

Called when field in that object is accessed through object.field or Object.get. When field is accessed through Object.getStrict or object["field"] call to this method will be omitted.

object _get_<field>()

Non-static May be null

Called when field <field> in that object is accessed through object.field or Object.get. When field is accessed through Object.getStrict or object["field"] call to this method will be omitted.

object _set(string key, object value)

Non-static May be null

Called when field in that object is set through object.field = value or Object.set. When field is set through Object.setStrict or object["field"] = value call to this method will be omitted.

object _set_<field>(object value)

Non-static May be null

Called when field <field> in that object is set through object.field = value or Object.set. When field is accessed through Object.setStrict or object["field"] = value call to this method will be omitted.
